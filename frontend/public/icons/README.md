# Iconos y logo

Coloca aquí los archivos de iconos y el logo del sitio.

- **logo.svg** o **logo.png** — Logo que se muestra en el menú (esquina superior izquierda).
- Otros iconos (favicon, redes, etc.) que quieras usar en la web.

Puedes reemplazar `logo.svg` por tu propio archivo; el header lo usará automáticamente.
