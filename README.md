# Psicoclub Monorepo

Aplicación moderna para Psicoclub reescrita desde WordPress usando:

- Frontend: Next.js 14 (App Router), Tailwind CSS, Framer Motion, Lucide React.
- Backend: Fastify (Node.js + TypeScript).
- Base de datos: PostgreSQL.
- Infraestructura: Docker y Docker Compose.

## Estructura

- `frontend`: Aplicación Next.js 14.
- `backend`: API Fastify en TypeScript.

## Comandos principales

- `npm install` (en la raíz) para instalar dependencias de todos los workspaces.
- `npm run dev:frontend`
- `npm run dev:backend`
- `docker-compose up --build`
